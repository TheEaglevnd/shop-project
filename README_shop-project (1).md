# Shop Project

A React + Vite e-commerce style project developed as part of the **Daneshkar Frontend Bootcamp**, showcasing component-based UI structure and SCSS styling.

## About
This project is built with a standard **React + Vite** setup and implements a shopping/e-commerce style page or app. The codebase follows a typical `src`/`public` structure and uses SCSS for styling.

## Tech Stack
- JavaScript / React
- SCSS
- HTML / CSS
- Vite + ESLint

## Project Structure
```
shop-project/
├── public/
├── src/
├── index.html
├── vite.config.js
├── eslint.config.js
├── package.json
└── README.md
```

## Getting Started
```bash
git clone https://github.com/TheEaglevnd/shop-project.git
cd shop-project
npm install
npm run dev
```
Then open the local address shown in your terminal (usually `http://localhost:5173`).

## Build for Production
```bash
npm run build
```
